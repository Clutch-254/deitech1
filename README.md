# Celestial Bodies Database Project

This repository contains the solution for the freeCodeCamp Relational Database course - Celestial Bodies Database project.

## College Event Feedback Analysis

This project analyzes student event feedback to uncover satisfaction trends and suggest improvements using survey data.

### Skills Gained:
- Data cleaning
- Sentiment analysis
- Survey insights
- Charting
- Natural Language Processing (NLP)

### Tools Used:
- Google Colab (for execution environment)
- pandas (for data manipulation)
- TextBlob (for sentiment analysis)
- seaborn (for data visualization)
- matplotlib (for plotting)
- WordCloud (for text visualization)
- Google Forms (CSV as input data source)

### Project Overview:
The project involves:
1.  **Data Loading and Cleaning**: Reading survey data (simulated CSV) and handling missing values and data type conversions.
2.  **Sentiment Analysis**: Applying TextBlob to open-ended feedback to categorize sentiment (positive, negative, neutral).
3.  **Data Visualization**: Generating various charts (e.g., overall rating distribution, sentiment distribution, ratings by year of study) and word clouds to visually represent insights.
4.  **Insights and Recommendations**: Deriving actionable insights and suggestions for improving future college events based on the analysis.
